Google fonts

How to use

<link rel="stylesheet" href="https://fonts.googleapis.com/css?family=font-name" >

Note: To use multiple fonts from google seperate each font with (&family="font-name")

Styling fonts

This font has the below properties

.h2-s1-p5{
font-family: "roboto", sans-serif;
text-shadow: 3px 3px 3px #ababab;
font-size: 1.2em;
}

Google font effects

There are various font effects that google provides such as.

  1. Fire effect
  2. Neon effect
  3. Outline effect
  4. Emboss effect
  5. Shadow-Multiple effect

How to use google effects

in the link tag after family=family-name add the following &effect=effect-name

To add multiple effects add a (|) pipe operator between each effect

Note: import the effects in head section of the HTML document and fonts can be imported to head sectino of HTML or top of CSS document
